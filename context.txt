1. WEBSITE OVERVIEW & ARCHITECTURE
========================================================================

The Focus Website is a high-impact, single-page product presentation built
to reflect the luxurious, quiet aesthetic of the Android application. It guides
the visitor from initial value-proposition awareness to an interactive
walkthrough of core software modules, culminating in an early sandbox access form modal.

*   1.1 RESPONSIVE GLASSMORPHIC HEADER
    - Fixed top positioning with heavy backdrop-blur filters (`backdrop-blur-md`).
    - Standard navigation anchors: Features (#features), Why Focus (#why-focus),
      Showcase (#showcase), and Reviews (#testimonials).
    - Custom brand identity embedding a micro-shadowed favicon and high-contrast
      typography.
    - Integrated premium CTA button: "Request Access" triggering the early stage sandbox intake form modal.

*   1.2 HERO LANDING BLOCK
    - Left column displays the critical trust badge ("100% Private & Fully Offline"),
      a high-prominence title featuring responsive line-height transitions, a highly
      persuasive core value narrative, and dual actionable CTAs.
    - Features a system summary metrics bar highlighting key values:
      "0% Trackers", "Early Access", and "100% Offline".
    - Right column houses a 3D-angled, interactive perspective-skewed mobile mockup 
      running the daily quote calligraphy home screen. Reacts to user hover states 
      via dynamic spring-loaded scale and tilt adjustments.

*   1.3 DYNAMIC PINS & FEATURES TIMELINE (`FeaturesSection.jsx`)
    - Pinned viewport utilizing GSAP's scroll-trigger pinning (`h-[480vh]` scroll track).
    - Features an ultra-premium **Interactive Progress Sidebar** (Desktop):
      * Semi-transparent custom timeline track (`#FFE7D0` at 10% opacity) with an
        orange-gradient progress line (`#FC6E20` to `#FC6E20/40`) filling dynamically
        according to precise scroll values.
      * Pulse-animated glowing active core rings that utilize Framer Motion's shared 
        layout capabilities (`layoutId="activeDotOuter"`). The glowing slider 
        literally slides up and down the track like a physical indicator.
      * Interactive glassmorphic tooltip pills displaying matching feature names when
        hovered or active.
      * Interactive dot click triggers calculating linear scroll offsets and initiating
        smooth window scrolls (`window.scrollTo`) to jump directly to specific features.
    - Features a **Smooth Snapping Mechanism** locking scroll states directly on
      individual feature cards (`1 / (N-1)` snapping increments) to prevent fast-scroll content skipping.
    - Responsive mobile override: Automatically swaps into a vertically stacked card structure
      for natural phone reading, disabling heavy viewport locking.

*   1.4 INTERACTIVE PRIVACY GRIDS (`WhyFocus.jsx`)
    - Displays card decks highlighting security benefits (zero-cloud tracking, room
      local sandboxing, premium composure, lightning responsiveness).
    - Utilizes a luxury layout with custom Lucide icon badges and grid structures.

*   1.5 SHOWCASE SLIDER (`CarouselSection.jsx`)
    - Responsive, swipe-capable preview carousel displaying native screenshots
      representing actual screens inside the application.

*   1.6 TESTIMONIALS & REVIEWS (`Testimonials.jsx`)
    - Glassmorphic cards with ratings, user bios, and descriptive reviews.
    - Built using custom card border designs and premium typography layouts.

2. CORE MOBILE APPLICATION FEATURES & MODULES
========================================================================

The website showcases the actual capabilities of the Focus Android App,
detailing these specific native software architectures:

*   2.1 DYNAMIC QUOTES ENGINE (HOME SCREEN)
    - Delivers high-fidelity curated quotes in a stunning calligraphy format.
    - Utilizes a unique, handcrafted calligraphy typeface (Google Font 
      "Playwrite GB S") exclusively for the core quote displays.
    - Allows swift sliding, sharing, and copying of quotes with a single tap.
    - Provides a functional "Refresh" trigger to fetch a new random piece 
      of wisdom from the localized catalog instantly.

*   2.2 CATEGORIZED EXPLORE HUB
    - Dive into tailored mental categories matching your current mood:
      Motivation, Calm, Focus, Hard Work, and more.
    - Features a responsive wrapping grid system optimized to display 
      perfectly across standard smartphones, tablet layouts, and 
      foldable devices without cutoffs.

*   2.3 ADVANCED SEARCH & FILTERING
    - Search through thousands of stored offline quotes by key terms, 
      concepts, authors, or categories.
    - Dynamic instant filtering ensures you find the exact words you need 
      to match your mental block.

*   2.4 OFFLINE FAVORITES VAULT
    - Bookmark the quotes that speak directly to your soul.
    - Keep them locally cataloged for permanent, fast offline reflection 
      whenever you lack an active internet connection.

*   2.5 DAILY MOOD TRACKING (ENFORCED SINGLE-LOG)
    - Keep a pristine log of your emotional states over the months.
    - Enforces a secure "once per day" logging policy. If you log a new 
      mood on the same day, it intelligently overwrites your previous choice.
      This guarantees your monthly analytics, averages, and charts remain 
      100% accurate and are never skewed by double-logs.

*   2.6 REFLECTIVE JOURNAL
    - An elegant personal diary system integrated directly inside the app.
    - Write daily reflections, goals, or notes with premium, book-grade 
      typography using the sleek "Literata" font family.
    - Track entries in a clean history format.

*   2.7 GAMIFIED BADGE & MILESTONE SYSTEM (ACHIEVEMENTS)
    - Earn and unlock customized aesthetic achievement badges for your consistency.
    - Gamifies your growth journey, encouraging daily check-ins.

*   2.8 SELF-DISCIPLINES STREAK TRACKER
    - Track consecutive days of logging into your journey.
    - Displays a highly motivating streak dashboard to keep your habits consistent.

*   2.9 INTELLIGENT NOTIFICATION SYSTEM
    - Automated daily notification alerts featuring inspiring quotes.
    - Solved status bar "White Box" bug by incorporating a standard-compliant, 
      crisp monochrome quotation mark outline icon (R.drawable.ic_quote_mark). 
      This ensures elegant, clean, high-contrast display badges.
    - Includes custom Quiet Hours settings to suppress notification triggers during 
      sleep block intervals (e.g. 10 PM to 7 AM).

3. VISUAL THEME, STACK, & DESIGN DESIGNATION
========================================================================

*   3.1 STACK DESIGNATION:
    - Framework: React 19 (React-DOM 19), initialized inside Vite (Vite v8).
    - Logic & Animation: GSAP v3 (ScrollTrigger) & Framer Motion v12.
    - Icons: Lucide React v1.
    - Styling: TailwindCSS v4.3.0 & Custom CSS utility tokens.

*   3.2 COLOR PALETTE & LUXURY AESTHETICS:
    - DarkBlack (#1B1B1B): The primary grounding body background, promoting
      peace and reducing ocular fatigue.
    - CreamBeige (#FFE7D0): The typography and border accent, introducing
      warm, comfortable bookish luxury.
    - VibrantOrange (#FC6E20): The central active branding, focus glow, and CTA
      theme color.
    - Includes ambient decorative glow backings (radial gradient overlays) with
      heavy blur scales (up to `blur-[150px]`) and pulsing animations to make the
      canvas feel response-driven and interactive.

*   3.3 SYSTEM TYPOGRAPHY:
    - Headings: Serif luxury ("Literata", styled font fallbacks), carrying
      sophistication and book-grade class.
    - Body Text: Clean geometric sans-serif, providing high-contrast readability.
    - Quotes & Special: Calligraphy cursive fonts highlighting handwritten presence.
    - Code & Statistics: Structured monospaced components.

4. PERFORMANCE & USER CONVENIENCE POLICIES
========================================================================

1.  🔒 ZERO CLOUD TRACKING & 100% PRIVATE:
    Unlike mainstream apps that sell your writing and mood profiles to 
    advertisers, Focus is completely offline-first. Your journals, logs, 
    names, and settings are saved locally on-device inside a secure, 
    sandboxed SQLite Room database. Your privacy is absolute.

2.  ⚡ LIGHTNING-FAST PERFORMANCE (NO WEB LAYERS):
    No webviews, no heavy cloud latency, and no bloated loading screens. 
    By pre-bundling TTF assets and optimizing the SQLite query paths, 
    the app performs with instant, butter-smooth 60fps animations.

3.  🧘 PROTECTS YOUR MENTAL PEACE:
    With custom "Quiet Hours" notification blocks, you control the noise. 
    Focus acts as a peaceful sanctuary, not another screen screaming 
    for your attention.

4.  🌐 SANDBOX ACCESS DISTRIBUTION:
    The website coordinates early sandbox distribution keys. All intake profiles
    are securely evaluated to maintain tranquility across our offline client database.
